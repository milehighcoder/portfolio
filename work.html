Scheduler App

This application saves events for each hour of the work day and features dynamically updated HTML and CSS powered by jQuery. The Moment.js library was used to display the date and time. Past time blocks are dark gray. Present time blocks are light gray. Future time blocks are white.

Weather App

This application showcases my first experience implementing a Third Party API into an application that I created from scratch and features dynamically updated HTML and CSS powered by Javascript. I used Bootstrap to design the UI. Users can check the weather conditions for any city and the browser will then store their searches.
