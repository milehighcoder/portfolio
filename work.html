Work

Here is a collection of my latest Github projects that feature mobile first responsiveness as well as a minimalist UI/UX approach.

Code Quiz

This application features a timed coding quiz that includes multiple-choice questions, a countdown timer that subtracts 10 seconds for every incorrect answer, a quiz score, and a save results feature that displays your latest high score. This app runs in a browser and features dynamically updated HTML and CSS powered by JavaScript code.

Repository

Scheduler App

This application saves events for each hour of the work day and features dynamically updated HTML and CSS powered by jQuery. The Moment.js library was used to display the date and time. Past time blocks are dark gray. Present time blocks are light gray. Future time blocks are white.

Repository

Weather App

This application is my experience implementing a Third Party API into an application that I created from scratch and features dynamically updated HTML and CSS powered by Javascript. I used Bootstrap to design the UI. Users can check the weather conditions for any city and the browser will then store their searches.
